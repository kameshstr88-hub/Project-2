# Text Limit Tracker

This project is a simple web-based text limit tracker built using HTML, CSS, and JavaScript.  
It monitors user input in real time and displays the remaining number of characters based on a predefined maximum limit.

---

## Objective

The objective of this project is to demonstrate:
- Real-time user input tracking
- Character counting logic
- DOM manipulation using JavaScript
- Basic user feedback through dynamic messages

---

## How to view
You can view with this link [https://kameshstr88-hub.github.io/Project-2/]

## Application Description

The application contains a textarea with a fixed maximum character limit.  
As the user types, the system calculates the number of remaining characters and updates the display dynamically.

When the character limit is reached, a warning message is shown to inform the user that no more input is allowed.

---

## Features

- Real-time character count update
- Fixed maximum character limit (200 characters)
- Automatic enforcement using `maxlength`
- Visual warning when limit is reached
- Clean and minimal user interface
- No external libraries or frameworks

---

## Technical Details

- **Maximum character limit**: 200
- **Event used**: `keyup`
- **Validation method**: Length-based character counting
- **Feedback mechanism**: Dynamic text and visibility control

---

## Technology Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
